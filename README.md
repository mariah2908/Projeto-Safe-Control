# Safe-Control
O Safe Control é uma plataforma digital desenvolvido para auxiliar empresas no controle de EPIS e Manutenção de Maquinários. O site busca cenbtralizar informações de segurança do trabalho de forma prática e organizada, permitindo acesso por computadores, celulares e tablets. Desenvolvido com HTML, CSS, Java Script e banco de dados MySQL, o projeto tem como objetivo reduzir riscos, otimizar processos e auxiliar no cumprimento das normas de segurança no ambiente de trabalho.
