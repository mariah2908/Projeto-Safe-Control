# Projeto-Safe-Control
Site desenvolvido para auxiliar empresas no controle de EPIS e manutenções de maquinários. O site busca centralizar informações de segurança do trabalho, permitindo maior oeganização, acompanhamento de prazos, controle de equipamentos e apoio as normas. A proposta é oferecer uma solução prática, acessível e moderna para reduzir riscos, otimizar processos e melhorar a gestão da segurança no trabalho. É uma plataforma web desenvolvida com HTML, CSS e Java Script, permitindo acesso por computadores, celulares e tablets. O sistema utilizará o MySQL para armazenar as informações oferecendo um solução online e prática visando a segurança.
